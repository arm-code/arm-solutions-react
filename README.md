# The Engineering Ledger

### How to configure the environment variables

```bash
cp .env.example .env
``` 
This project uses the following environment variables:
- VITE_SUPABASE_URL=https://mmi...
- VITE_SUPABASE_ANON_KEY=eyJhbGciO...

We need to set the values of these variables in the .env file before running the application. You can get these values from your Supabase project settings.


### How to run the application

```bash
npm run dev
```
